SQUY 2D Plateformer

About the Project

SQUY is an exciting 2D platformer where your goal is to navigate through challenging levels while avoiding spinning wheels. Precision and timing are key as you jump, dodge, and maneuver your way to the finish. Enjoy vibrant graphics and engaging gameplay in this thrilling adventure.

Current Features

  • Multiple Levels: Diverse and progressively challenging levels.
  • Spinning Wheel Obstacles: Various spinning wheels placed in harder levels to avoid.
  • Hidden jump plateforms: you can go through the grey plateforms to help reach certain places
  • Tutorial Mode: Introductory levels or guides to help new players learn the mechanics.

Technologies Used

  • Unity Game Engine
  • C# Programming Language
  • Photoshop for asset creation
