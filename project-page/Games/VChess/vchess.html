vChess

About the Project

VChess is a project that replicates the logic of the classic game of chess. Designed to showcase smooth gameplay mechanics, VChess focuses on accurately representing the core mechanics of chess without the need for full visual representation. This project is ideal for those interested in studying and experiencing the strategic elements of chess.

Current Features

  • Classic Chess Mechanics: Faithful reproduction of traditional chess rules and logic.
  • Move Validation: Ensure all moves adhere to standard chess rules.
  • Minimalistic Design: Focus on the core mechanics with a clean and simple interface.
  • Highlight Legal Moves: Visual aid to show all possible legal moves for each piece.

Technologies Used

  • Unity Game Engine
  • C# Programming Language
  • Photoshop for asset creation
