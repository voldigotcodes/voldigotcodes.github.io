AERON

About the Project

Aeron is an electrifying game set in a neon, cube-filled world where players glide left and right to dodge obstacles. With its dynamic levels and pulsating soundtrack, Aeron delivers a heart-pounding experience that challenges your reflexes and precision. Immerse yourself in this visually captivating adventure and see how far you can go in the ever-changing landscape of Aeron!

Current Features

  • Dynamic Levels: Constantly changing environments that keep the gameplay exciting.
  • Neon Aesthetics: Bright, colorful graphics that create a visually striking experience.
  • Intuitive Controls: Simple and responsive controls for smooth gameplay.
  • [In development] Endless Mode: Infinite levels for an unending challenge.
  • Obstacle Variety: Different types of blocks and hazards to dodge, increasing the difficulty.

Technologies Used

  • Unity Game Engine
  • C# Programming Language
  • Photoshop for asset creation
