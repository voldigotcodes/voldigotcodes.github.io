Floppy bird


The Floppy Bird project is an homage to the popular and challenging mobile game, Flappy Bird, which gained widespread acclaim for its simple yet addictive gameplay. The objective of this project was to recreate the essence of Flappy Bird while adding a personal touch to the game development process. The replica features a bird character controlled by tapping on the space bar of your keyboard, navigating through a series of pipes with the aim of achieving the highest score possible.

It uses multiple different algorithms for query optimisation, aka to optimise any request done on any database to bring the result out in the least possible time.
