Hotel Booking system

About the Project

The Hotel Booking App is an Android application designed for seamless hotel reservations. Utilizing an SQL database for data storage, the app supports two types of accounts: Manager and Client. Managers oversee hotel operations and manage employees, while Clients can browse and book hotels effortlessly.

Current Features

  • Account Types: Two distinct user roles - Manager and Client, each with specific functionalities.
  • Manager Dashboard: Tools for managing hotels, employees, and overseeing bookings.
  • Client Profiles: Users can browse hotels, make reservations, and manage their bookings.
  • Booking System: Easy-to-use interface for clients to search, book, and confirm hotel reservations.
  • SQL Database: Robust data storage solution for managing user information, bookings, and hotel details.
  • Availability Calendar: View hotel availability and room options in a calendar format.
  • Notifications: Real-time updates for booking confirmations.
  • Search Filters: Advanced search options to filter hotels based on location, price, rating, etc.
  • Employee Management: Managers can add, remove, and manage hotel staff through the app.
