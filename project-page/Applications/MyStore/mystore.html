myStore

About the Project

myStore is designed as a customer review add-on for online e-commerce websites. Built with PHP, HTML, and MySQL, this application allows users to create, read, update, and delete reviews, providing valuable feedback for products. It seamlessly integrates into existing e-commerce platforms, enhancing user engagement and improving the overall shopping experience.

Current Features

  • Create Reviews: Users can submit new reviews for products they have purchased.
  • Read Reviews: Display existing reviews for products, including ratings and comments.
  • Update Reviews: Users can edit their previously submitted reviews to reflect updated opinions.
  • Delete Reviews: Option for users to remove their reviews if needed.
  • Rating System: Allows users to rate products on a scale, enhancing the review's usefulness.

Technologies Used

  • PHP
  • XAMPP
  • MySQL
  • HTML
  • CSS
