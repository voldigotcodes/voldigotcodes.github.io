Flappy Bird Replica Project

About the Project

The Floppy Bird project is an homage to the popular and challenging mobile game, Flappy Bird, which gained widespread acclaim for its simple yet addictive gameplay. The objective of this project was to recreate the essence of Flappy Bird while adding a personal touch to the game development process. The replica features a bird character controlled by tapping or clicking, navigating through a series of pipes with the aim of achieving the highest score possible.

Original Features

  • Flappy bird character controlled by space bar
  • Randomly generated pipes as obstacles
  • Scoring system saved on the device so players can find their highscore at all time
  • Scoring system based on successfully passing through gaps

Added Features

  • Randomly generated coins allowing user purchase power ups and [in progress]
  • Randomly generated power ups that increase the score faster and gives you less pipes to dodge as a break

Technologies Used

  • Unity Game Engine
  • C# Programming Language
  • Photoshop for asset creation
